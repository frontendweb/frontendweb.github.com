Deploying your file

For this assignment you'll be modifying your index.html page, adding a css directory, and adding a main.css file

  1. Create a directory (aka folder) with the name "css" in the same directory as your index.html page
  2. Create a file called main.css and put it in the "css" directory
    • /{user-name}.github.com
      • README.md
      • index.html
      • /css New!
        • main.css New!
  3. In index.html:
    • Add a table with some interesting data with at least 4 rows and 4 columns. If you can't think of anything, separate arachnids into their respective subspecies...just kidding...you can come up with something!
    • Separate your homepage using <div>
    • Add a class to one of the <div>
    • Add an id to one of the <div>
  4. In main.css:
    • Add a background-color to your <body>
    • Style (however you want) all your <p>
    • Style ONE of the <div> by using a class selector
    • Sytle ONE of the <div> by using an id selector
    • Highlight one of your cells in the table
  5. Commit and sync this to your GitHub page
