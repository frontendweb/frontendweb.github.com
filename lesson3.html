Development Tools

Popular tools include Firebug in Firefox, Chrome Developer Tools in Chrome, and Firefly in Opera.

In Chrome F12 to open Developer Tools:

Ctrl+O: Open file

Ctrl+Shift+O: Find function

Ctrl+Shift+F: Find variable

Deploying your file

You can log in to your account from github.com

Go to the directory that you want to create your file in.

Click on the plus sign next to the directory name

Name your file. If you want to create a directory you can just type '/' and it will auto create a directory for you

Make the desired changes to your file

Scroll to the bottom to the page, add a commit message and press "Commit New File"
