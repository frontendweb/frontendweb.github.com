Lesson 6 September 24, 2013

Goals For This Lesson

  • Demonstration/Walkthrough
    • Using Emmet (formerly Zen Coding)
    • Bootstrap 3 (formerly Twitter Bootstrap)
      • Adding a navigation bar
      • Add a Jumbotron
      • Using the Bootstrap Grid system
        • Add subheadings
    • Using Handlebars for templating (time permitting)
    • Using Firebase to store our JSON (time permitting, last on the list because our company blocks it)

What your personal directory should look like at the end of this lesson:

  • /{user-name}.github.com
    • README.md
    • index.html
    • /css
      • main.css
    • /js
      • main.js
    • /lib
      • /bootstrap
        • All the bootstrap content

Demonstration/Walkthrough

  1. Download Bootstrap

  2. Open up a new JSFiddle

    • Select jquery 1.9.1 under Frameworks & Extensions

    • The following are CDNs (Content Delivery Networks)

      • Add //netdna.bootstrapcdn.com/bootstrap/3.0.0/css/bootstrap.min.css to External Resources

      • Add //netdna.bootstrapcdn.com/bootstrap/3.0.0/css/bootstrap-theme.min.css to External Resources

      • Add //netdna.bootstrapcdn.com/bootstrap/3.0.0/js/bootstrap.min.js to External Resources

  3. Add a navigation bar
  4. Add a Jumbotron
  5. Add subheadings using the Bootstrap grid system
  6. Add images to those subheadings and make the images responsive
