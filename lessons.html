GitHub Pages

Go to GitHub.com and setup an account.

Create a repository using your {user-name}.github.com as the name of the repository where {user-name} is your username

For example my user name is frontendweb so my repository name would be frontendweb.github.com

Download and install the GitHub client for Windows or for Mac

Create a file called index.html and add some HTML to it. You can put anything in it just like we learned in Codecademy but here's an example below:

															
	<html>
		<body>
			<p>Hello World!</p>
		</body>
	</html>

Now using what we learned in class, update that index.html page with the following info:

  • A heading with your name
  • An unordered list with your 3 favorite hobbies
  • An ordered list with your top 3 songs including genre
  • An image of your favorite animal
  • A paragraph with what you hope to get out of this class (as long or as short as you like)
  • A self rating of what your knowledge of this topic was prior to coming to this class: beginner, intermediate, advanced
Congrats! You've deployed your first website
