GitHub at Work September 30, 2014

Here are the steps to edit the files when you’re at work. The firewall blocks normal GitHub usage so the steps from the tutorial won’t work inside our network.

 

1.       Login to github

2.       Go to your repository. For example if you used awesome321 as your username and setup the repo following the tutorial it would be: https://github.com/awesome321/awesome321.github.com

3.       At the top of that page there should be a “+” icon next to your repository name: awesome321.github.com/+

4.       Click on the “+” and you should be brought to a “New file” page. In the input box where it reads “Name your file…” you can enter a filename (eg index.html)

5.       Make all your changes here, then at the bottom of the page you can fill in the section “Commit new file” the same as in the tutorial and your page should deploy the same
