Create an Account and Repository

Go to GitHub and create an account

Select a plan (we're using the free one) and sign up.

Click on the + symbol in the top right hand corner and create a new repository

Name your repository your {USERNAME}.github.com
This is VERY IMPORTANT, if this does not match your site will NOT DEPLOY

Awesome! Your repository is ready to go! A repository is like a shared folder between anyone that has access to it (think Dropbox). We'll be storing all our files here. At this point verify your email or else your site will also NOT DEPLOY

Next, download the GitHub Client for Windows or Mac and follow the instructions for installation

Deploy a File

Once you're setup, login using your GitHub account from earlier

Click on dashboard

Click on the + sign on the top left corner

Select Clone, we don't want to Create since we already created a repository earlier

You should see the repo you created earlier, select it

Click Clone

You'll want to clone your repo to your GitHub directory, by default it's in User/Documents/GitHub

Congratulations, you now have a copy of your repository you set up earlier

Now we want to open this folder up in the Window's Explorer and copy our index.html from your homework into this directory

Go back to the GitHub Client, and click on uncommitted changes

The GitHub Client knows that you modified that directory! The text highlighted in green are additions, for now since it's a new file, everything will be green. Later on when you edit your files you'll see red as well for anything that you removed

Add a short description of what we're doing here so if we want to look at this change in the future we'll know what happened and click "Commit to Master"

Great your changes are now saved on your computer! But we still need to sync up with that folder "in the cloud..." so go ahead and click "Sync"

Excellent you should be all synced up now!

You can now go to your website by entering your URL: {USERNAME}.github.com. If this is your first time deploying, the 404 page can show up for up to 10 minutes before you can actually see your site. If it still does not work after that amount of time, check out the FAQ or shoot me a message. Good luck!
