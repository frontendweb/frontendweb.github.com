Create an Account and Repository

The two most important steps for your site to work:

  • Your repository name has to be your {USERNAME}.github.com exactly. For example if your GitHub username is awesome321, your repo name has to be awesome321.github.com
  • You have to verify your email before you deploy your index.html. If you forget to do this you will have to redeploy the index.html file after you've verified

Go to GitHub and create an account

Select a plan (we're using the free one) and sign up.

Click on the + symbol in the top right hand corner and create a new repository

Name your repository your {USERNAME}.github.com
This is VERY IMPORTANT, if this does not match your site will NOT DEPLOY

Awesome! Your repository is ready to go! A repository is like a shared folder between anyone that has access to it (think Dropbox). We'll be storing all our files here. At this point verify your email or else your site will also NOT DEPLOY

Deploy a File

Once you're setup, login using your GitHub account from earlier

Click on the + symbol next to the repository name

Enter a file name, in this case index.html

Add some changes to the file

Scroll down to the commit message

Add a description of what your changes are doing and commit them using the default shown

Your file is now added!

Type {USERNAME}.github.com where {USERNAME} is your username and enjoy your newly minted webpage!

Deploy your css file

Click the "+" sign just like you did when you created your index.html file

Type "css/" into the input box. The "/" tells GitHub you are making a directory

Enter a file name, in this case main.css

Add some changes to the file

Scroll down to the commit message and add a description of your changes

Your file is now added! Now we just need to link up your index.html file

Go back to your index.html and hit edit shown under the red arrow here

Add your link, it should look something like this if you named you file the same as mine: main.css

Scroll down to the commit message and add a description of your changes

Your main.css file should be linked and your directory should look like this.

You're now ready to add some style to your page!

Add the Bootstrap CSS library to your website

Click the "+" sign just like you did when you created your index.html file

Type "lib/" into the input box, then "bootstrap/". The "/" tells GitHub you are making a directory. Then name your file bootstrap.min.css

Copy the code from the Bootstrap CDN and paste it into this file. It should all show up in one line since its minified (aka packaged to optimally transfer through the internet)

Commit the file just like you did with your index.html and main.css files. Make sure the description is relevant for your own reference in the future

Now the last step is to add a reference in our index.html file. Your href should point to the directory that you saved bootstrap.min.css in. In this case "/lib/bootstrap/bootstrap.min.css". Alternatively, you can directly link to the CDN as show here. You'll only need one of these references

You can now reference Bootstrap CSS classes while in your index.html!

Add the jQuery library to your website

Use the plus sign like we've done in the past to add a file

Type in "lib/" followed by "jquery/" followed by "jquery.min.js"

Go to jQuery CDN and copy+paste the code, then commit

Go back to your root folder and create another file by pressing the "+". This time, type "js/" followed by "main.js"

Add the jQuery ready block we learned in class to prep your JS file for jQuery then commit the file

Go back to the root directory and click on index.html so we can edit it

Add either a reference to your local copy of the file or a remote reference. In the example below I'm referencing the local file with the reference to the remote file being in the comment block. You will only need one of these lines. It's up to you which one you prefer.

Also you'll need to add a reference to main.js so that your JS code will work. Make sure this line is below the jQuery line or else you won't be able to use the library in main.js

You now have a link to the jQuery library and you can start using it on your own!
