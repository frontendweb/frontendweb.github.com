Create an Account and Repository

The two most important steps for your site to work:

  • Your repository name has to be your {USERNAME}.github.com exactly. For example if your GitHub username is awesome321, your repo name has to be awesome321.github.com
  • You have to verify your email before you deploy your index.html. If you forget to do this you will have to redeploy the index.html file after you've verified

Go to GitHub and create an account

Select a plan (we're using the free one) and sign up.

Click on the + symbol in the top right hand corner and create a new repository

Name your repository your {USERNAME}.github.com
This is VERY IMPORTANT, if this does not match your site will NOT DEPLOY

Awesome! Your repository is ready to go! A repository is like a shared folder between anyone that has access to it (think Dropbox). We'll be storing all our files here. At this point verify your email or else your site will also NOT DEPLOY

Deploy a File

Once you're setup, login using your GitHub account from earlier

Click on the + symbol next to the repository name

Enter a file name, in this case index.html

Add some changes to the file

Scroll down to the commit message

Add a description of what your changes are doing and commit them using the default shown

Your file is now added!

Type {USERNAME}.github.com where {USERNAME} is your username and enjoy your newly minted webpage!
