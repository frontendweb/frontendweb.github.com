If you missed Tuesday’s class you can follow these steps to deploy your main.css file (I’ll try to post a follow-up with screenshots by this Friday to the tutorial page as well):

·         Press the “+” sign in your github directory just like you did when you created your “index.html” file

·         Type in “css/”, it’ll create a directory named “css” for you

·         Type in “main.css” (you can technically name this anything as long as you have the “.css” extension)

·         Make your changes here and commit them just like you did in your index.html file

·        

o    <link rel=”stylesheet” href=”css/main.css”>

o    This links your index.html file to you css file so be sure to name it whatever you named you css file
